Initialization
==============

To initialize the Kinesis JS SDK you need to add the following Javascript and CSS code to your HTML

    <!-- kinesis stylesheet -->
    <link rel="stylesheet" type="text/css" href="http://cdn.kinesis.io/kinesis.css">
  
    <!-- kinesis js sdk -->
    <script src="http://cdn.kinesis.io/kinesis-js-sdk.min.js"></script>
    <script>    

      // initialize kinesis
      var kinesis = new Kinesis; 

    </script>
